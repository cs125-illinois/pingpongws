<Lead>

Typescript support for wrapping WebSocket connections to enable bidirectional ping-pong messages.

</Lead>

```bash
npm i @cs125/pingpongws
```

## Use

Visit the documentation [here](https://github.com/cs125-illinois/pingpongws/).

<PingPonger />

## Demo

To see the ping-pong wrappers in action, run `npm start` and then examine both the terminal console (server pong messages) and browser console (client ping messages).
The examples also show how to use the `filterPingPongMessages` wrapper to filter ping-pong messages from your application code if you don't want to handle them.
